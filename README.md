# mssql-export-tools
Microsoft SQL Server object exporting tools

## Export-Dtsx

Export SQL Server's SSIS-packages (package model) to \*.dtsx files.


### Usage

```powershell
PS C:\> Export-Dtsx -Server contoso.com -Output ~\dtsx\contoso
```


### Help

```powershell
PS C:\> Get-Help Export-Dtsx -Full
```
